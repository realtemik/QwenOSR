# Qwen_Voice_Chat
Qwen Voice Chat — локальный AI-ассистент с поддержкой записи системного звука, распознавания речи через Whisper и анализа текста через Ollama/Qwen. Приложение позволяет вести чат с ИИ, отправлять голосовые фрагменты с компьютера, получать потоковые ответы в реальном времени и работать полностью локально без облачных сервисов.
